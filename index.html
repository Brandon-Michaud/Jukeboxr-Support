How to report a bug

Please include the following:

  • iOS version (e.g., 17.6), device model (e.g., iPhone 14)
  • App version/build (e.g., 1.0 (3))
  • Steps to reproduce, expected vs. actual result
  • Any screenshots or screen recordings

Common questions

  • How do I join the beta? If you have a TestFlight invite link, install TestFlight from the App Store and open the link on your iPhone.
  • Where can I find the privacy policy? See the Privacy Policy.
